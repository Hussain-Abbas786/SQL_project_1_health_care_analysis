# Healthcare Analysis using MySQL

## Overview
This project analyzes healthcare patient and appointment data using MySQL.

## Technologies Used
- MySQL
- MySQL Workbench

## Features
- Patient analysis
- Appointment trend analysis
- Missed appointment analysis
- Department-wise insights
- Repeat patient analysis
- No-show rate analysis

## SQL Concepts Used
- Joins
- GROUP BY
- Aggregate Functions
- CTEs
- Window Functions
- RANK()
- LAG()

## Conclusion
This project demonstrates how SQL can be used for healthcare data analysis and extracting operational insights.
